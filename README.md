# StratoView

StratoView is an interactive web application that visualizes atmospheric conditions on a global map. It integrates weather radar, lightning activity, aircraft positions, and additional weather data such as temperature and forecasts into a unified, Google Earth–style interface. The app emphasizes real-time situational awareness by allowing users to explore how weather systems and air traffic coexist spatially.

How to run locally:
    In project directory run:
        pip install -r requirements.txt
        streamlit run app.py
